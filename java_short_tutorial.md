#### interface

interface关键字用来定义一个接口类型，接口可以认为是一个完全抽象的类型，表示实现这个接口的那些类的共同行为。

接口的*方法（methods）*默认是公开抽象（public和 abstract）的。

接口的*常数（constants）*默认是public, static and final的。
